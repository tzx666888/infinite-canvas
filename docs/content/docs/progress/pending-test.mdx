# 待测试

- 我的素材：分类筛选已在素材页直接展开，内置场景、人物、男模、女模、商品、服装、道具、动作姿势、镜头构图、风格参考、品牌素材、提示词和其他，并支持自定义分类；画布、生图工作台、视频创作台和提示词库保存素材时会先弹出分类菜单。需验证分类选择与取消保存、新增、编辑、搜索、筛选、画布选择器、旧素材补默认分类及导入导出。
- v3.0.25 电商分镜视频：独立关键帧按 Beat 使用 12 宫格的对应单格锁定构图、人物服装和动作，原始素材只锁定人物/商品身份；整片视频直送已批准关键帧，禁止未规划的换装、桌子、额外衣物、商品提前出现和口播改写；需用含真人、服装和商品的完整分镜核对首帧、中段动作、人物/商品一致性、口播自然度和可见口型。
- 配置与用户偏好：每个模型渠道新增 OpenAI / Gemini 调用格式选择，默认 OpenAI，切换默认格式时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到“模型”Tab 选择。
- Gemini 调用格式：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 调用格式时先给出不支持提示，仍需使用 OpenAI 兼容渠道。
- Nano Banana 2 原生尺寸：画布的 `gemini-3.1-flash-image` 已拆分为 1K、2K、4K 三个公开模型名，直接传递对应原生分辨率与 14 种官方比例，保留上游原图，不再在浏览器内拉伸或锐化；未开放 0.5K。
