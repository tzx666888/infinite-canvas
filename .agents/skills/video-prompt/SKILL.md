# Grok 视频提示词

## 当前模型
- grok-imagine-video-1.5-fast：文生视频或 1 张参考图图生视频，6/10/15 秒。
- grok-imagine-video-1.5-preview：1-7 张参考图，多参考图视频，6/10 秒。
- grok-imagine-video-1.5-1080p：必须 1 张参考图，6/10 秒，原生 1080p。

不得使用其他上游模型名，不得擅自换模型、时长、比例或参考图模式，不得用 4K/8K 冒充原生输出。

## 输出
只输出 90-180 词英文单段 prompt，不加标题、解释、Markdown、中文或复杂时间轴。用 then、while、as 串成一条可执行主线。

## 编写顺序
1. 当前时长与比例。
2. 主体、动作和一个连贯场景。
3. 景别、运镜、光线和节奏。
4. 人物与产品身份、现实尺度和物理连续性。
5. 当前参考模式约束。
6. 简洁质量词和 Negative prompt。

## 参考模式
- 文生视频：不提不存在的参考图。
- 单图：把参考图作为首帧和身份权威，只增加合理运动，保留构图、主体位置、人物、产品、服装、logo 和标签位置。
- 多参考图：按输入顺序把第一张作为主要身份锚点，后续图仅补充外观、场景或镜头方向；不同人物或产品绝不融合。最终视频是干净全屏镜头，不显示原宫格、边框、编号或拼图。
- 审阅分镜图永远不能直接传给视频模型；只使用 metadata 为 keyframe 的干净关键帧。

## 稳定性
- 保持人物五官、年龄段、发型、服装和产品轮廓、现实尺寸、比例、材质、部件、logo、标签位置。
- 用手、身体、桌面、家具、门、地面或环境锁定尺度。
- 单个关键帧视频任务只执行对应 beat，不重复整条视频计划。
- 动作、接触、重量、阴影、反射和透视必须物理合理。
- 末尾约束身份漂移、产品变形、人体畸变、重复主体、网格、字幕、水印、虚假文字与虚假声明。

## 安全
遵守 shared-commerce。泳装、内衣或贴身服装只采用明确成年人的非色情时尚编辑/商品目录语境；不突出私密部位，不用窥视、恋物或年龄模糊镜头。
