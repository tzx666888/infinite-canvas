# 提示词优化 Skill

## 触发条件
客户在画布上选中节点，点击润色菜单里的「✨ 优化提示词」，再点 Agent 执行。

## 完整工作流

### 第一步：读取输入
- 读取客户在提示词框里写的文字
- 读取连接到当前节点的参考图（如有）

### 第二步：优化提示词
将粗糙描述转化为高质量生图提示词。

#### 优化规则
1. 保留用户原始意图，不改变主题和核心元素
2. 补充画面细节：光线、色调、构图、景别、材质质感
3. 增加专业摄影/设计术语提升画面质量
4. 输出中文，方便用户阅读和修改
5. 控制在 100-200 字
6. 不要加任何解释，只输出优化后的提示词
7. 禁止使用 beautiful / amazing / epic / stunning / gorgeous / incredible 这类空泛词

### 第三步：回填结果
用 `canvas_update_node` 或 `canvas_update_node_text` 将优化后的提示词回填到节点的 prompt 字段，供客户确认后生图。

## 输入输出示例

输入：「一双黑色运动鞋放在桌子上」

优化后：「一双黑色网面跑步鞋斜放于浅灰色水泥台面上，三分之一俯视角度，鞋头朝向画面右侧。白色中底清晰可见，红色后跟拉环形成色彩点缀。柔和侧逆光从左上方打入，在鞋面网眼上形成细腻光影层次，台面上投射出淡淡的接触阴影。背景为虚化的暖灰色调，浅景深突出产品质感。」
