# 提示词优化

## 触发
客户选择“优化提示词”时执行。先读取节点类型、当前提示词、参考图和当前模型配置。

## 图片模式
输出 120-280 字中文提示词，只输出最终结果。

必须覆盖：
- 保留主体、动作、场景、风格、构图、数量、版式和比例。
- 补充必要的镜头、景别、光线、材质、色彩、空间关系和景深，不堆砌空泛画质词。
- 参考图是人物、产品或角色身份权威；锁定五官、发型、服装、轮廓、现实尺寸、颜色、材质、部件、logo 和标签位置。
- 产品大小按参考图与品类动态推断，用手、身体、桌面、家具、门或环境作为尺度锚点，不使用固定通用占比。
- “生成 N 张”表示 N 个独立任务；只有客户明确要求宫格、海报或信息图时才写多面板版式。
- 只保留清楚可见的原有文字。看不清时保持标签版式，不猜测文字，不添加价格、角标、认证或水印。

## 视频模式
输出 90-180 词英文单段 prompt，只输出最终结果。

必须覆盖：
- 使用当前模型、时长、比例和参考图数量，不擅自换模型或时长。
- 一条清晰动作主线，具体写主体、动作、场景、运镜、光线、节奏和物理约束。
- 单图保持首帧构图；多参考图按输入顺序锁定身份和镜头，不融合不同人物或产品，不显示网格。
- 保持人物、产品、服装、现实尺寸、logo 与标签位置，不写模型不支持的 4K/8K。

## 安全与真实性
- 不编造功效、认证、成分、价格、折扣、销量、评价、医学结果或不可验证的前后对比。
- 人物保持自然骨骼、面部和手指。
- 泳装、内衣或贴身服装只采用明确成年人的非色情时尚编辑/商品目录语境；姿势自然、服装不透明，不突出私密部位，不使用窥视、恋物或年龄模糊表达。
- 禁止 beautiful、amazing、epic、stunning、gorgeous、incredible 等空泛词。

## 回填
用 canvas_update_node_text 回填最终提示词，不额外创建说明节点，不声称已经生成图片或视频。
